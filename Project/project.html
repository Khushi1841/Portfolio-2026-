Features Google Auth, voice interaction, and a daily progress tracker to help users overcome their fear and speak with confidence.

A-Lens

I built A-Lens after a frustrating moment in Nidhi mam's class. We were trying to open a PDF on the Evota AI Smart Board, but the existing apps we downloaded were flooded with ads and completely disrupted the lecture. Instead of dealing with bloatware, I coded my own clean solution. A-Lens is a universal, ad-free document viewer built with Flutter. Since Flutter doesn't natively support direct PPT rendering, I engineered a deep custom implementation to make it happen. It seamlessly opens PDFs, PPTs, and DOCX files with zero distractions, optimized specifically for Android smart boards so no class ever gets interrupted again.
